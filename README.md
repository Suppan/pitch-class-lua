# **pitch-class-lua**
simple set class calculator written in Lua

after installing Lua (ie via homebrew: brew install lua)

enter midi values [0-127] in terminal: 
```
lua pc-info.lua 60 64 55
```
result:
```
chord: 60 64 55
name: 3-11
prime: 037 iv: 001110
```
